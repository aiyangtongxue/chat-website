# VideoHub - 全网视频解析平台

## 项目简介
VideoHub 是一个便捷的在线视频解析工具，支持主流视频平台的视频解析和播放。用户只需粘贴视频链接，即可实现视频的无障碍观看。

## 功能特点
- 🎯 支持多平台视频解析
  - 哔哩哔哩（Bilibili）
  - 腾讯视频
  - 爱奇艺
  - 优酷
  - 芒果TV
  - YouTube
- 🚀 简洁直观的用户界面
- ⌨️ 支持回车快捷键触发解析
- 📱 响应式设计，完美支持移动端
- 🎨 现代化深色主题界面
- 🔄 实时视频解析播放

## 使用说明
1. 打开网站，在输入框中粘贴需要解析的视频链接
2. 点击"解析播放"按钮或按回车键
3. 等待视频加载完成即可观看
